v-for:列表渲染
Vue官方文件
可用在頁碼或其他應用

可以接受整數作為渲染的條件

渲染“陣列”

陣列是元素的集合排列而成

v-for="(value陣列數值, index索引值) in 陣列

只要傳入合法變數即可以取出陣列中相對應數值

  • Array index: {{index}} Array value: {{item}}
渲染“物件”

物件是key：value的方式組成

v-for="(value屬性值, key屬性鍵值, index索引值) in 物件

只要傳入合法變數即可以取出物件中相對應數值

  • Object index: {{index}} Object key: {{key}} Object value: {{val}}
